## KenshiLib_Examples dependencies
This repo contains the dependencies required to compile the [KenshiLib_Examples](https://github.com/BFrizzleFoShizzle/KenshiLib_Examples/) repo.

The setup script sets up environment variables that are used in the `KenshiLib_Examples` projects.

## Setup
Clone the repo (do NOT download as .zip as that won't download LFS files)

Run Setup.bat in the directory (it should automatically run as administrator)
